# media-club UI
Vite + Rolldown implementation for UI

## Commands
Run local development server  
`npm run dev`

## Application Setup
1) Initiate backend services. Read [here](https://github.com/Bmoel/media-club/tree/main/apps/media-club-api) for more details
2) Run `npm run dev`
3) Navigate to `http://localhost:5173/` to see the application
